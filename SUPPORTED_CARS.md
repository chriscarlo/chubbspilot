# Openpilot Supported Car Fingerprints

This document lists all car fingerprints supported by openpilot. These fingerprint values can be used for testing with the `FINGERPRINT` environment variable.

**Usage Example:**
```bash
FINGERPRINT=HONDA.HONDA_ACCORD ./launch_openpilot.sh
```
